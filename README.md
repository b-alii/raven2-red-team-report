This repository contains my final report for an authorised Red Team Exercise performed against the Raven 2 vulnerable virtual machine in a controlled laboratory environment.

- Attack Surface Enumeration
- Vulnerability Analysis
- Initial Access
- Post-Exploitation
- Privilege Escalation
- Objective Collection
- Reporting

- Report Writing

- Raven 2 (VulnHub)
